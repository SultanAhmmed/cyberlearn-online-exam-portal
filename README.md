# CyberLearn - Online Examination Portal

CyberLearn is a modern PHP-based Online Examination Portal created for educational and cyber security learning purposes.

This project is specially designed for students learning:
- Web Application Security
- Ethical Hacking
- Penetration Testing
- Secure PHP Development
- Authentication Systems
- Database Security

The portal starts as a front-end project and will gradually evolve into a complete cyber security practice lab.

## Current Features
- Modern responsive UI

